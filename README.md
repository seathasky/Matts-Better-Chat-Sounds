Matt Better Chat Sounds is a lightweight addon that plays sound alerts for Guild, Whisper, Party, and Raid chat.

It supports:
- Modern WoW (Midnight)
- Mists of Pandaria Classic
- Classic Era/Hardcore
- TBC Classic

---

## Features

- Plays a sound when a message appears in supported chat channels.
- Lets you enable or disable sounds per chat type (Guild, Whisper, Party, Raid, and related channels).
- Includes an ignore list so messages containing specific words or phrases do not trigger a sound (for example: `Questie`, `Nova World Buffs`).
- Simple configuration through the options window, opened by slash command or minimap icon.

---

## Usage

- Type `/mbcs` in chat to open the options.
- Or click the minimap icon to open settings.
- In the options window, use the Ignore Words section to add, remove, or clear filtered terms.
